Attributes

The href attribute of the anchor tag, tells you where the anchor leads.
You can also use these attributes to identify your elements

    Ways to identify an element using attributes
  • ID - unique way to identify an element
  • Class - a way to identofy multiple elements

Different Types of Selectors

Simple Selectors

  • Tag/Element
    • uses the tag name to identify the element
    • this will affect all elements with that tag name
  • Class
    • uses the class name to style a group of elements
    • denoted by .className
  • ID
    • Uses a unique id
    • denoted by #id
  • Pseudo-class
    • specific keyword at the end of the selected used to style elements when it is at a certain state
    • : active, :hover, ect...
  • Pseudo-element
    • used to style specific parts of an element
    • insert content before or after an element
    • style first letter or line of an element
    • ::before, ::after, ::firstline, ::firstletter

Combinator Selectors

  • Child Descendant
  • Sibling
  • Attribute selectors
  • Multiple selectors

Wild card selector

  • Just the *
  • Styles everything, all elements with the given rules
